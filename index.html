Why TeaWind

CSS without the stress.

Whenever styling gets overwhelming — take a sip, write a class, move on.

⚡️

~1ms Initialization

Map() cache — every class resolved once, retrieved instantly. Hundreds of elements, still milliseconds.

🔄

MutationObserver

A DOM watchdog fires on every new node. React renders? Styled instantly. No useEffect. No setup.

🧹

Self-Cleaning DOM

Every chai- class disappears after processing. Open DevTools — only clean inline styles remain.

🎨

Built-in Palette

Every color has a -light and -dark variant. Handpicked hex codes. No memorization needed.

🧠

Smart Dynamic Values

chai-p-24 → padding: 24px. Numbers auto-convert to px. Strings like full pass through unchanged.

📦

Zero Dependencies

No PostCSS. No Webpack. No config. One import 'teawindcss' and you're live.

Quick Start

Up in 30 seconds.

Three ways to use TeaWind — pick what fits.

Step 1

Install

$ npm install teawindcss
Step 2

Import Once

import 'teawindcss' ;
// auto-initializes. done.
Step 3

Write Classes

chai-flex chai-p-16
chai-bg-blue-dark
chai-rounded-xl

Manual Control — brew()

for React / dynamic HTML
import { brew } from 'teawindcss' ;
// Manually process DOM after dynamic renders
brew () ;
